# angirashukla.github.io

Personal academic website for Angira Shukla, built with [Quarto](https://quarto.org).

## Edit & preview locally
```bash
quarto preview
```

## Publish
```bash
quarto publish gh-pages
```
(Or use the Render / Publish buttons in RStudio.)

## Add photos
Drop web-friendly JPGs into `photos/` — the gallery on the Photography page
rebuilds itself from that folder on the next render.

## Swap the headshot
Replace `assets/headshot.jpg` with your own image (same filename).
